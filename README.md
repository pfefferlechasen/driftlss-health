# Driftless Health — Healthcare Niche Site

Therapy-focused version of the Driftless Agency site, targeting ABA clinics, OT/PT practices, SLP centers, and sensory gyms.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Stack

- Next.js 15 (App Router)
- Tailwind CSS v4
- Framer Motion
- Lucide Icons

## Deploy to Vercel

1. Push to a new GitHub repo
2. Connect to Vercel
3. Deploy — zero config needed
